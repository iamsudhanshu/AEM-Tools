AEM Server Comparison

This is a tool to compare two AEM servers

How to Use This Tool

×

Getting Started

1. Upload or paste JSON data from two AEM servers you want to compare.

2. Choose what to compare using one of the buttons:

  • Compare Bundles: Compares OSGi bundles between servers
  • Compare Packages: Compares installed packages between servers

Understanding the Results

The comparison table uses icons to show differences:

  • Different Values
  • Only in First JSON
  • Only in Second JSON
  • Identical Values

Different values are highlighted in red for easy identification.

Additional Features

  • Getting bundle JSON: http://localhost:4502/system/console/bundles.json
  • Getting package JSON: http://localhost:4502/crx/packmgr/list.jsp
  • Upload JSON files or paste content directly
